# Компиляция в exe

Создаём виртаульное окружение командой:

* windows
    * `python -m venv .venv`
* linux
    * `python3 -m venv .venv`

Далее заходим в него

* windows
    * `python .venv\Script\activate`
* linux
    * `python3 .venv/bin/avtivate`
